🎵 日語音調（高低アクセント）

日語使用「音高重音」：每個音節（拍）有高（H）或低（L）兩種音高。東京標準語有四類音調型：

類型說明例詞音高示意
平板型第一拍低，之後全高，後面助詞也保持高みず（水）、はし（箸）低→高高高…
頭高型第一拍高，之後全低（下降在第1拍後）あめ（雨）、かみ（紙）高→低低低…
中高型第一拍低，中間某拍後下降たまご（卵）、こころ（心）低→高→降↓
尾高型第一拍低，之後全高，助詞後才下降おとこ（男）、はし（橋）低→高高→助詞↓

💡 同形假名的音調不同，意思就不同！例如：はし — 「橋」（尾高型 LHH）vs「箸」（平板型 LHH助詞不降）vs「端」（頭高型 HL）

🔤 清音・濁音・半濁音

在基本清音的基礎上，添加符號可改變發音：

種類符號變化說明例子
清音無基本無聲（或輕聲）輔音か・さ・た・は行
濁音濁點 ゛聲帶振動，變為有聲輔音か→が（k→g）、さ→ざ（s→z）、た→だ（t→d）、は→ば（h→b）
半濁音半濁點 ゜は行變為雙唇音 pは→ぱ（h→p）、ひ→ぴ、ふ→ぷ、へ→ぺ、ほ→ぽ

💡 連濁（れんだく）：兩個詞合併時，後詞第一個音節的清音有時自動變濁音。例：はな（花）＋み（見）→ はなみ（花見）；やま（山）＋かわ（川）→ やまがわ（山川）

⏸ 促音（っ）— 雙子音

促音「っ」（小寫）代表停頓一拍，相當於後面輔音被「加倍」。發音時在促音處短暫停頓（約一拍的時間），再發後面的音。

例詞羅馬字中文說明
きってkitte郵票t+t，停頓後發t音
ざっしzasshi雜誌ss，停頓後發sh音
にっきnikki日記kk，停頓後發k音
まってmatte等一下tt，停頓後發t音

💡 促音如果寫錯位置，意思完全不同！如：きて（來）vs きって（郵票）

📏 長音（伸ばす音）— 拉長母音

日語母音可以延長，相當於發音時間加倍（約兩拍）：

母音平假名寫法片假名寫法例詞
あ段長音あ→ああ（如：おかあさん）ア→アーおかあさん（母親）、ラーメン
い段長音い→いい（如：おにいさん）イ→イーおにいさん（哥哥）、タクシー
う段長音う→うう（如：くうき）ウ→ウーくうき（空氣）、スーパー
え段長音え→えい/ええエ→エーえいご（英語）、ケーキ
お段長音お→おう/おおオ→オーおおきい（大）、コーヒー

💡 片假名的長音用「ー」（長音符）表示，羅馬字通常用重音符號或雙母音表示（如：ā、ū、ō）

🔀 拗音（ようおん）— 複合音

拗音是將い段假名與小寫「ゃ・ゅ・ょ」組合，讀成一拍（不是兩拍）：

組合發音例詞意思
きゃ・きゅ・きょkya / kyu / kyoきゅうり小黃瓜
しゃ・しゅ・しょsha / shu / shoしゃしん・しょくじ照片・餐飲
ちゃ・ちゅ・ちょcha / chu / choおちゃ茶
にゃ・にゅ・にょnya / nyu / nyoにゅうがく入學
ひゃ・ひゅ・ひょhya / hyu / hyoひょうじゅん標準
みゃ・みゅ・みょmya / myu / myoみょうじ姓氏
りゃ・りゅ・りょrya / ryu / ryoりょこう旅行

👃 撥音（ん）— 鼻音 N

ん是日語唯一一個單獨成音節的輔音，發音隨後接音節變化：

後接音ん的發音例詞
ば・ぱ・ま行（雙唇音前）m（雙唇鼻音）さんぽ → sam-po
な・に行（齒齦音前）n（齒齦鼻音）さんない → san-nai
か・が行（軟顎音前）ng（軟顎鼻音）さんかく → saŋ-kaku
文末・母音前・ら行前ñ（鼻化元音）ほん → hoñ

💡 ん獨立成一拍，所以「さんぽ」是3拍（sa-n-po），「かんじ」是4拍（ka-n-ji）
